Installing snowpark-connect-deps-2
==================

Installing `snowpark-connect-deps-2` from the main channel can be achieved by:

```
conda install snowpark-connect-deps-2
```

It is possible to list all of the versions of `snowpark-connect-deps-2` available on your platform with `conda`:

```
conda search snowpark-connect-deps-2
```
